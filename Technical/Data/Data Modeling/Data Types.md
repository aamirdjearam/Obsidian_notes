
### Numeric Types

- INT - Standard 32-bit integer (Whole Number)
- SMALLINT - Smaller 16-bit integer
- TINYINT - Very Small 8-bit integer
- MEDIUMINT - Medium Sized 24-bit integer
- BIGINT - Large 64-bit integer
- DECIMAL - Exact fixed-point number (good for money)
- NUMERIC - Same as DECIMAL (exact precision)
- FLOAT - Approximate single-precision floating point
- DOUBLE - Approximate double-precision floating point
- BIT - Stores bit values (Either 0 or 1 or bit fields)
